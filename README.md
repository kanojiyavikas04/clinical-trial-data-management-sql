# Clinical Trial Data Management System (SQL Server)

This project demonstrates a Clinical Trial Data Management (CDM) system built using SQL Server.
It includes patient enrollment, study visits, laboratory results, and adverse event tracking.

## Technologies Used
- SQL Server
- SQL Server Management Studio (SSMS)

## Key Features
- Patient enrollment management
- Visit and laboratory data tracking
- Adverse event monitoring
- Data quality validation queries
